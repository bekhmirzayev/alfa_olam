# Alfa Olam - Digital Parenting Platform

## Overview
**Alfa Olam** is a digital parenting platform designed to help parents understand and manage the digital lives of "Generation Alpha" (children born after 2010). The platform provides scientific insights, data analytics, and a balance simulator to help parents find the right mix of technical control and trust.

## Features
- **Scientific Insights**: Key facts and research about Generation Alpha's digital habits.
- **Data Analytics**: Visualizations of global smartphone access and app usage trends.
- **Educational Topics**: 15 key topics covering technical basics, monitoring, safety, and psychology.
- **Balance Simulator**: An interactive tool to visualize the impact of control vs. trust on the parent-child relationship.

## Technologies Used
- **HTML5**: Semantic structure.
- **Tailwind CSS**: Utility-first styling for responsiveness and design.
- **Chart.js**: Interactive bar and line charts.
- **Plotly.js**: Radar chart for the balance simulator.
- **Font Awesome**: Icons for visual enhancement.
- **Google Fonts**: "Plus Jakarta Sans" for typography.
